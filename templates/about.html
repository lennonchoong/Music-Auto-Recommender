About This Project

This project was built during the COVID-19 social lockdown. Flask is used as the web framework. Python is used for back-end framework and scripts. YouTube Links provided through webscraping using requests module and Spotify Links were provided through Spotify Web API and spotipy module. Recommendations are based off genre and example artist.
